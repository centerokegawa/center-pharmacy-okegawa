# センター薬局 桶川店 ホームページ運用ガイド

## 画像の差し替え方法

トップページの一番上に表示されるメイン画像（AIで生成した仮の画像）を、実際の薬局の外観写真や店内の写真に差し替える手順です。

1. **写真の準備**: 実際に使用したい写真（例：`exterior.jpg` や `photo.png`）をご自身のパソコンで用意します。
2. **画像の保存**: 用意した写真を `center-pharmacy/images/` フォルダの中にコピー（保存）します。
3. **HTMLの書き換え**: 
   - `index.html` をテキストエディタ（またはメモ帳、VS Code等）で開きます。
   - `49行目` 付近にある以下のコードを探します：
     ```html
     <img src="images/hero.png" alt="センター薬局 桶川店の外観">
     ```
   - この `images/hero.png` の部分を、保存した新しい画像の名前（例：`images/exterior.jpg`）に書き換えて保存します。

これでブラウザを再読み込みすると、実際の写真に差し替わります！

## テキストや営業時間の変更
- `index.html` 内の文字を書き換えるだけで、画面上の表示もすぐに変わります。
- 営業時間の変更も同様に `index.html` の表（`<table class="hours-table">` 付近）を書き換えることで修正可能です。
